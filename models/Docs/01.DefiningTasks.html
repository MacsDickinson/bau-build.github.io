Require() .DependsOn("groom_myself", "walk_dog") .Task("turn_off_alarm") .Do(() => Console.WriteLine("Turned off alarm. So tempted to snooze...")) .Task("make_coffee").DependsOn("turn_off_alarm") .Do(() => Console.WriteLine("Made coffee.")) .Task("groom_myself").DependsOn("make_coffee") .Do(() => { Console.WriteLine("Brushed teeth."); Console.WriteLine("Showered."); Console.WriteLine("Shaved."); }) .Task("walk_dog").DependsOn("make_coffee") .Do(() => Console.WriteLine("Walked the dog.")) .Run();

Tasks can be defined in any order, can depend on any number of other tasks DependsOn("foo", "bar"), can be referenced multiple times with multiple calls to Task("baz") and can have multiple actions assigned to them with multiple calls to Do().

TIP: you can write any C# you like in your baufile. The scriptcs #load and #r features are both supported for loading scripts and referencing assemblies.
